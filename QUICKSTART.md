# Quick Start Guide

## Get Started in 3 Steps

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Application
```bash
npm run dev
```

### 3. Open in Browser
Navigate to: **http://localhost:3000** (or http://localhost:3001 if 3000 is in use)

---

## Your First Story

### Step 1: Create Your First Chapter
1. Click **"New Chapter"** on the Dashboard
2. Fill in:
   - Episode Number: 1
   - Title: "The Beginning"
   - Arc: "Introduction Arc"
   - Status: "In Progress"
3. Start writing in the content area
4. Click **Save** (or wait 30 seconds for auto-save)

### Step 2: Add Main Characters
1. Click **"Characters"** in the navigation
2. Click **"Add Character"**
3. Fill in character details:
   - Name, age, role
   - Description and appearance
   - Personality traits
   - Abilities (comma-separated)
   - Backstory
4. Click **"Save Character"**

### Step 3: Build Your World
1. Click **"World"** in the navigation
2. Add elements by clicking:
   - **+ Location**: Add cities, landmarks
   - **+ Magic System**: Define your power system
   - **+ Faction**: Create organizations
   - **+ Lore**: Add background history
3. Fill in details and save

### Step 4: Continue Writing
1. Return to **"Chapters"**
2. Click on any chapter to edit
3. Keep writing your story!

### Step 5: Export Your Work
1. Click **"Export"** in the navigation
2. Select chapters to export
3. Choose format (Text, Markdown, or JSON)
4. Click the export button
5. Your file will download automatically

---

## Tips

- **Auto-save**: Your chapters auto-save every 30 seconds
- **Word Count**: Track your progress with automatic word counting
- **Status Tracking**: Mark chapters as draft, in-progress, or completed
- **Organization**: Use story arcs to group related chapters
- **Backup**: Export regularly or copy the `story-data` folder
